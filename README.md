# Stochastic Laplacian Heaviside Contraction Code

The contraction code that we use for meson observables.

## Documentation

Documentation is generated by Doxygen. Run `doxygen` in the source and then
view `doxygen/html/index.html` with your browser.

## Dependencies

In order to build the program, you need the following:

- [c-lime](https://github.com/usqcd-software/c-lime) library
- Boost library
- CMake and ctest
- Doxygen
- Eigen library
- g++
- git
- googletest
- HDF5 command line utilities (`h5diff`)
- HDF5 library
